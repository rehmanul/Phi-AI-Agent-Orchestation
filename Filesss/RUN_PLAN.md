# Investment Sales BD System - Run & Test Plan

## Goal
Run the complete system end-to-end, host it locally, and capture screenshots to understand the UI and identify areas for improvement.

## Steps

### Step 1: Install Dependencies
- Install Python requirements for backend
- Check if frontend dependencies exist (Next.js setup)

### Step 2: Start Backend API
- Start FastAPI server on port 8000
- Verify health endpoint responds

### Step 3: Start Frontend
- Start Next.js dev server on port 3000
- Verify frontend loads

### Step 4: Navigate to Application
- Open browser to http://localhost:3000/bd/intake
- Navigate through all pages:
  - /bd/intake (Data Intake)
  - /bd/targets (Target Review)
  - /bd/call-list (Call List)

### Step 5: Capture Screenshots
- Screenshot each page
- Document what's visible and functional

### Step 6: Analysis
- Review screenshots
- Identify what works
- Identify gaps or improvements needed

## Expected Outcomes
- System running locally
- Screenshots of all pages
- Analysis of UI state and functionality
